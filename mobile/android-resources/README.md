# Android resources

Drop your high-res source assets here. Capacitor / Bubblewrap pick these up:

- `icon-foreground.png` — 432×432 transparent foreground for adaptive icon
- `icon-background.png` — 432×432 solid background (or use `#0f1318` flat)
- `icon-monochrome.png` — 432×432 white-on-transparent for Android 13+ themed icons
- `splash.png` — 2732×2732 centered logo on `#0f1318`

If you skip these, Capacitor will fall back to `../public/pwa-512x512.png`.
